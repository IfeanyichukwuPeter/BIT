## [1.0.0] - 2026-02-26

### Added
- Initial website launch
- Public-facing pages (Home, About, Programs, Contact)
- Admin dashboard with authentication
- Question & Answer system
- Token-based admin authentication
- RESTful API backend
- Comprehensive test suite (97.89% coverage)
- Mobile-responsive design
- Event showcase (past, ongoing, upcoming)
- Leadership team profiles
- Social media integration

### Technical Highlights
- Modular server architecture (controllers, routes, middleware, utils)
- File-based JSON data storage
- Express.js backend
- Jest + Supertest for E2E testing
- Dynamic component loading (nav/footer)

### Security
- Token-based authentication for admin operations
- Protected endpoints for PATCH and DELETE operations
- Environment variable configuration
- Input validation and sanitization
